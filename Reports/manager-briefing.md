# Manager Briefing

A security monitoring exercise was successfully conducted using Wazuh SIEM in a controlled virtual lab environment. During the activity, simulated reconnaissance actions were performed from a Kali Linux system to test detection and monitoring capabilities.

Wazuh successfully detected and displayed security-related events, allowing the SOC workflow to be demonstrated from detection to documentation. Alerts were reviewed, categorized, and documented according to incident response procedures.

Evidence including screenshots, monitoring logs, and workflow diagrams was preserved for reporting purposes. The exercise demonstrated practical SOC operations including monitoring, alert triage, incident response, and reporting using industry-standard cybersecurity tools and techniques.
